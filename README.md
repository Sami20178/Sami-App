# Sami-App

Sami App als **Samibookkonsole Frontend-Prototyp** mit vielen Menüs, A-Z Launcher und erweiterten Funktionen.

## Features

- Große Navigation: Home, A-Z Apps, Projekte, Terminal, Kalender, Benachrichtigungen, Spiele, Apps, Tools, Media, KI/Modu, Start, Lernen, Store, Einstellungen, Account, Hilfe, Abmelden.
- A-Z App Launcher mit alphabetischer Filterung.
- Projekte-Bereich zum Hinzufügen eigener Projekte.
- Terminal-Demo mit Befehlen: `help`, `date`, `clear`, `open ai`, `open home`, `open projects`.
- Kalender-Ansicht mit heutigem Datum.
- Benachrichtigungen mit "als gelesen markieren".
- AI-Ansicht mit dynamischen Antworten (intent-basiert).
- Suchfeld mit Bereichs-Navigation und Schnellkacheln.
- Theme-Umschaltung (Dark/Light) + Live-Uhr in der Statuszeile.

## Start

1. `index.html` im Browser öffnen.
2. Menü oder Schnellkacheln nutzen.
3. In `KI / Modu` Fragen stellen.
4. In `A-Z Apps` per Buchstaben filtern.
5. In `Terminal` Befehle testen.
